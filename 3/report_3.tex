\section{結論}
本実験を通じて、常微分方程式の数値解法における各手法の特性を理解し、刻み幅や安定性が解の精度に与える影響を確認した。特にカオス現象の数値計算では、初期値の微小な差異が結果に大きな影響を及ぼすことを観察し、数値解法の慎重な選択と検証の重要性を認識した。
